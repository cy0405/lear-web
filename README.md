# learn-web
学习项目 单机版 业务逻辑主要根据煎架的博客项目为主。
纯练习知识点的一个手脚架 除主程序外（gin+gorm）其余皆为可插拔插件

程序涉及到的技术点如下
------------
- goroutine，channel
- kafka
- redis
- gin
- gorm 2.0
- es
- monogodb
- bigcache

------------
性能分析
- gc
- pprof
- trace
- godebug
- gops
- go-torch

------------
 部署
- jenkins
- docker
- k8s


------------
 监控
- Prometheus
